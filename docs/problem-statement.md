# Problem Statement

Agent handoffs are often treated as chat text, but handoffs carry more than
text. They can carry:

- trust;
- provenance;
- authority;
- approval context;
- memory scope;
- stale assumptions;
- tool output that may be untrusted.

The research problem is to define small, testable contracts that make those
properties explicit enough to validate.

This project starts with local synthetic examples. It does not claim to define a
global standard.

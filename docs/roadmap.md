# Roadmap

## v0.1

- Dependency-free envelope model.
- Deterministic verifier.
- Basic provenance and authority checks.
- Synthetic examples and tests.

## v0.2

- JSON loader and CLI.
- Report artifacts.
- More transfer scenarios: tool output, memory write, approval, OCR transcript.

## v0.3

- Compatibility examples with `ai-agent-handoff`.
- Cross-run diff for envelope changes.
- Audit-chain checks.
- First long-horizon semantic drift scenario.

## Later

- Optional signature/hash support.
- Adapter examples for common agent runtimes.
- Integration notes for `agentic-security-harness`.
- Research tracks: cross-ecosystem transfer integrity, perception boundaries,
  capability binding, semantic drift of refusal boundaries, and tamper-evident
  audit trails. See [research-roadmap.md](research-roadmap.md).
